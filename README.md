# 🚀 AWS EC2 Portfolio Website

A live portfolio website deployed on AWS EC2 using Amazon Linux and Apache.

## Live Demo
http://52.14.111.64/

## Project Overview
This project was built to learn the fundamentals of cloud deployment and remote server management. I launched a Linux-based virtual server on AWS EC2, configured networking and security access, connected remotely through SSH, installed Apache, and hosted a custom HTML/CSS/JavaScript portfolio site live on the internet.

## Tech Stack
- AWS EC2
- Amazon Linux
- Apache Web Server
- SSH
- HTML
- CSS
- JavaScript
- IAM

## Features
- Live website hosted on AWS
- Custom portfolio landing page
- Basic JavaScript interactivity
- Remote file editing and deployment through SSH
- IAM-based access instead of root-only usage

## What I Learned
- How to launch and manage cloud servers
- How to configure HTTP and SSH access
- How web servers deliver site files
- How to edit and deploy changes on a live remote machine
- Why IAM and MFA matter for security
